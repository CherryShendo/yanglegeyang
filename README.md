# 羊了个羊无限道具教程（此教程仅针对电脑版）

①登陆PC微信

②左下角小程序进入羊了个羊

③打开微信左下角设置/文件管理/打开文件夹，打开后返回WeChat Files文件夹爱

④进入Applet文件夹

⑤进入文件最后为`c970a9`的文件夹，打开15（比如说今天是9月15日，就是15；如果是9月16日，就是16）

⑥替换压缩包里的文件
